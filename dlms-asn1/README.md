# dlms-asn1

ASN.1 BER/DER encoding for DLMS/COSEM ACSE layer.

## Overview

Implements the ACSE (Association Control Service Element) messages:
- **AARQ** — Association Request
- **AARE** — Association Response
- **RLRQ** — Release Request
- **RLRE** — Release Response

As defined in IEC 62056-53 for the application layer of DLMS/COSEM.

## Dependencies

- `dlms-core`
